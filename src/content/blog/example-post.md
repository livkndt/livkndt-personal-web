## Getting Started

To create a new blog post:

1. Create a new markdown file in the `src/content/blog/` directory
2. Add frontmatter with the required fields
3. Write your content in markdown
4. Commit and push to your repository

## Features

- **Markdown Support**: Write your posts in markdown
- **Mobile-Friendly**: Edit posts from your phone using Decap CMS
- **SEO Optimised**: Automatic meta tags and Open Graph support
- **Dark Mode**: Automatic dark mode support

Happy blogging!
